# Website LLM

An AI Engineering project to build a website generation model from open-source web development data.

## Project Goal

This project aims to:

- Crawl web development resources
- Build a clean training dataset
- Fine-tune an open-source coding LLM
- Deploy a website generation assistant

## Project Structure

```
website-llm/
│
├── crawler/
├── datasets/
├── docs/
├── notebooks/
├── scripts/
├── tests/
```
